Weather Analysis by Carlos Villanueva Jr

For this project, our goal was to look at weather data and analyze the changes we identified when getting close to the equator.

In order to complete this assignment, we had to pull data from the OpenWeatherMap API to get the dataset that includes more than 500 cities

Once we were able to create the dataset, we were able to use Matplotlib in order to make our plots to show the weather vs latitude. The list of factors we observed include the following: temperature, cloudiness, wind speed, and humidity. On this site we have included source data, and the visualizations that were made for the analysis, and these visualizations include explanations for their correlations.
